# 🚀 Supply Auctions on Stellar (Soroban Smart Contract)

## 📌 Project Description
Supply Auctions is a decentralized smart contract built on the Stellar Soroban platform. It enables users to create and participate in auctions for goods or services in a transparent, trustless, and secure environment.

This project demonstrates how blockchain technology can be used to simplify and automate supply chain bidding processes without intermediaries.

---

## ⚙️ What It Does
- Allows users to create auctions for supply items
- Enables participants to place bids on listed items
- Automatically tracks the highest bid and bidder
- Allows auction owners to end auctions securely
- Provides real-time access to auction data

---

## ✨ Features
- 🛒 Create auctions for any supply item
- 💰 Real-time bidding system
- 🔒 Secure and decentralized logic using Soroban
- 👤 Ownership-based auction control
- 📊 Transparent bid tracking
- ⚡ Fast execution on Stellar network
